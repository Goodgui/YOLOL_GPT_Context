}}<section begin=summary/>The socket board provides an access panel to connect [[Generator_(Assembly)|generators]] to the electricity and data network (cables).

Note that one socket board per 100 tier 1/2 [[generator unit]]s (or 80 tier 3 generator units) is required, if cables are connected via a cube heat sink a socket board is still required to export power.<section end=summary/>

== Basic Usage ==

* Transforms Local Power into '''Power'''.
* Connects cables.
* Distributes '''Power''' to a [[Data networks|data network]].
* Can distribute a maximum of 100,000 '''Electricity''' per second. 
* Doesn't get any benefits from enhancers.
