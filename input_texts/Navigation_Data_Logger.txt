}}<section begin=summary/>When Capital ship is ready for fast travel it can fast travel to destinations saved into Navigation Chips with Navigation Data Logger. Saved Navigation Chip are shown in Fast Travel Terminal as available fast travel destinations when having the chip in inventory. Navigation Data Logger is unlocked in "Advanced Crafting Tree". <section end=summary/>

== Functionality ==

Travel to the location in the universe you wish to save, insert empty Navigation Chips into the chip slots, and select the amount of chips you want to save the location to.

# Navigation Data Logger needs to be fully assembled and connected to an electricity source in order to function
# Insert empty Navigation Chip(s) into the provided slots in order to save the current location into the Navigation Chips
# Select the amount of chips you wish to save the location to, and give a name to the Navigation Chips 
# Saving a location to a Navigation Chip requires 6000 electricity per second for 10 minutes
